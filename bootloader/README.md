# Tchux Bootloader


## Memory Map

```
+------------------------------------+ 0x100000 (1 MB) \n
|                                    | \n
|              FREE                  | \n
|                                    | \n
+------------------------------------+ 0xC0000 (768 KB)
|          BIOS (256 kB)             |
+------------------------------------+ 0xA0000 (640 KB)
|        Video Memory (128 kB)       |
+------------------------------------+ 0x9FC00 (~639 KB)
| Extended BIOS Data Area (639 kB)   |
+------------------------------------+ 0x7E00  (~511 KB)
|                                    |
|    **BOOT SECTOR LOAD AREA**       |
|        Bootloader @ 0x7C00         |
|   (512 bytes - boot sector size)   |
|                                    |
+------------------------------------+ 0x7C00  (31.25 KB)
|                                    |
|        **GDT** (0x7C0B)            |
|                                    |
+------------------------------------+ 0x7C0B
|                                    |
|                                    |
|     Kernel loaded at 0x1000        |
|                                    |
|                                    |
+------------------------------------+ 0x11000 (64 KB + 4 KB)
|                                    |
|              FREE                  |
|                                    |
+------------------------------------+ 0x500  (1280 bytes)
| BIOS Data Area (256 bytes)         |
+------------------------------------+ 0x400  (1024 bytes)
| Interrupt Vector Table (1 kB)      |
+------------------------------------+ 0x0    (memory start)
```
